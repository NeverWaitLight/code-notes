# Epic 3 视频优化与代理处理

目标：在上传视频时生成体积更小的代理视频文件，确保帧数严格对齐，然后对代理视频进行 HLS 分片处理，优化存储空间和播放性能。该 epic 结束后，系统将使用代理视频进行 HLS 分片，而非原始视频。

## Story 3.1 代理视频生成与 HLS 分片优化

As a 使用者,
I want 上传视频时自动生成体积更小的代理视频并用于 HLS 分片，
so that 可以节省存储空间并保持播放质量。

### Acceptance Criteria

1. 上传完成后，先生成代理视频文件，代理视频需保持与原视频帧数严格对齐。
2. 代理视频文件体积需比原视频文件体积大幅减少（目标：减少 50% 以上）。
3. 对代理视频文件执行 HLS 分片处理，而非原始视频。
4. 原始视频文件保留在本地存储，代理视频文件也保存到本地存储。
5. 其它上传、列表、播放、删除逻辑保持不变。
6. 删除视频时需同时删除原始视频、代理视频和 HLS 分片文件。
